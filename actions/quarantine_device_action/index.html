Version: 0.0.1

Actions used to quarantine a device

Requests

/action/quarantine/device/by_hostname

Quarantines a device by its hostname

payload: object 

{
    "target": {
        "device": {
            "device_id": "{cba1fc32-cc02-4f98-b210-35ecad2b5ea1}",
            "hostname": "testdevice.local.com"
        }
    }
}
target: object
device: object

A 'Device' Target MUST contain at least one property.

hostname: string

A hostname that can be used to connect to this device over a network

idn_hostname: string

An internationalized hostname that can be used to connect to this device over a network

device_id: string

An identifier that refers to this device within an inventory or management system

payload: object 

{
    "status": 200
}
status: integer , x ∈ { 102 , 200 , 400 , 401 , 403 , 404 , 500 , 501 , 503 }

/action/quarantine/device/by_mac_address

Quarantines a device by its MAC address

payload: object 

{
    "target": {
        "mac_addr": "B8:CA:3A:77:A5:AE"
    }
}
target: object
mac_addr: string

A Media Access Control (MAC) address - EUI-48 or EUI-64 as defined in [EUI]

payload: object 

{
    "status": 200
}
status: integer , x ∈ { 102 , 200 , 400 , 401 , 403 , 404 , 500 , 501 , 503 }
